# 🚀 Govt Jobs Website

A simple and user-friendly website that helps students find **Government Job Opportunities** based on their qualification.

---

## 📌 Features

* 🎓 Filter jobs by qualification:

  * 10th
  * Inter
  * Diploma
  * ITI
  * Degree
  * B.Tech

* 📊 Job details include:

  * Job Role
  * Organization
  * Age Limit
  * Salary
  * Apply Link
  * Official Website

* 🌐 Direct apply links to official portals

---

## 🚀 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Enter your name and select qualification
4. Click submit to explore jobs
